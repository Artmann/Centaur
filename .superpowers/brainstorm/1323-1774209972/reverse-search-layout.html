Reverse Search Layout

Ctrl+R opens this full-screen overlay on top of the terminal

Terminal Window — Reverse Search Active

❯ gg  8 / 142

Key behaviors

  • Ctrl+R — Opens overlay, focus goes to search input
  • Type to filter — Fuzzy matches highlighted in accent color + bold
  • Up/Down arrows — Navigate the results list
  • Enter — Execute the selected command immediately
  • Escape — Close overlay, return to terminal
  • Results anchored to bottom — Most recent commands closest to the input bar (like fzf)
  • Match counter — Shows "matched / total" in bottom-right
